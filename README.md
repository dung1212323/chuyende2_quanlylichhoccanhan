# 📅 StudySchedule App

## Hệ Thống Quản Lý Lịch Học Cá Nhân

---

## 📋 Giới Thiệu

**StudySchedule** là ứng dụng giúp người dùng quản lý lịch học cá nhân một cách hiệu quả và tiện lợi.
Người dùng có thể dễ dàng thêm, chỉnh sửa, xóa và theo dõi lịch học của mình mọi lúc mọi nơi.

Dự án được xây dựng nhằm phục vụ mục đích học tập và phát triển kỹ năng lập trình **Flutter + Backend API**.

## 📦 Cài Đặt

### 🔧 Backend

```bash
git clone https://github.com/your-repo/study-schedule.git
cd study-schedule/backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```

---

### 📱 Frontend (Flutter)

```bash
cd study-schedule/frontend
flutter pub get
flutter run
```
